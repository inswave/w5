CSS API

CSS Class를 핸들링 할 때에, .set("class", className)을 사용하다 보면 불편한 점이 많습니다. .set("class", className)의 경우 기존 class에 새로운 class를 추가하는 개념이 아닌 override의 개념이기에 이전에 지정했던 CSS Class들이 모두 없어지기 때문입니다.
그래서 W5 Grid는 CSS Class를 보다 수월하게 조작하기 위해서 .addClass(), .hasClass(), .removeClass(), .toggleClass() API를 제공합니다.

.addClass(className)

선택한 W5 Grid Parts 요소의 class에 인자로 넘겨준 class를 추가합니다.
매개변수의 data type은 Stirng으로 white space를 구분으로 여러개의 class를 추가할 수 있습니다.

Example

grid.row(0).addClass("bgRed");      // 첫번째 row에 bgRed 클래스를 추가합니다.
grid.row(0).addClass("bd1 f-bold"); // 첫번째 row에 bd1과 f-bold 클래스를 추가합니다.

.hasClass(className)

W5 Grid Parts Select Method로 선택한 요소가 해당 class를 가지고 있는지 여부를 반환합니다.
white space를 구분으로 여러개의 class 지정하지 못합니다.

Example

// cell(0, 3)이 "f-italic"란 클래스를 갖고 있으면 return true, 아니면 false
grid.cell(0, 3).hasClass("f-italic");
// colModel ID가 col2인 column이 "f-bold"란 클래스를 갖고 있으면 return true, 아니면 false
grid.col("col2").hasClass("f-bold");

.removeClass(className)

선택한 W5 Grid Parts 요소의 Class중 className에 해당하는 class가 있으면 해당 class만 제거합니다.
매개변수의 data type은 Stirng으로 white space를 구분으로 여러개의 class를 추가할 수 있습니다.

Example

// 세번째 row의 "f-italic"이란 class를 삭제한다.
grid.row(2).removeClass("f-italic");
// colModel ID가 col2인 column의 "f-bold"란 클래스 삭제한다.
grid.col("col2").removeClass("f-bold");

.toggleClass(className, switch)

선택한 W5 Grid Parts 요소의 Class중 className이

없으면
.addClass(className)합니다.
있으면
.removeClass(className)합니다.

두번째 인자 switch

switch는 필수 인자는 아닙니다.
switch의 값이 true이면 className을 addClass(className)하고, false면 removeClass(className)합니다. switch를 지정하면 아래와 같 javascript code를 작성할 수 있습니다.

var addOrRemove = grid.cell(0, 0).get("data");

// addOrRemove가 true이면 bgBlue class를 추가하고,
// false이면 bgBlue class를 제거합니다.
grid1.col(1).toggleClass("bgBlue", addOrRemove);

이 코드는 다음과도 같습니다.

var addOrRemove = grid.cell(0, 0).get("data");

if ( addOrRemove ){
    grid.col(0).addClass("bgBlue");
} else {
    grid.col(0).removeClass("bgBlue");
}
