CSS API

You will find difficulties when handling CSS Class with .set("class", className). .set("class", className) does not add a new class to the existing, instead it overrides the existing CSS Class so the existed CSS Class disappears.
In W5 Grid, to handle CSS Class easier, provides .addClass(), .hasClass(), .removeClass(), .toggleClass() API.

.addClass(className)

On the selected W5 Grid Parts element, it adds a class of which passed on as a parameter.
The data type of the parameter is a String and can add multiple classes by white space separator.

Example

grid.row(0).addClass("bgRed");      // Add bgRed class to the first row.
grid.row(0).addClass("bd1 f-bold"); // Add bd1 and f-bold classes to the first row.

.hasClass(className)

It returns whether the selected W5 Grid Parts element has the class of which is passed as a parameter or not. .hasClass( className) only checks the class of which is passed as a className.

Example

// if the cell(0, 3) has the "f-italic" class, it returns true, if else false.
grid.cell(0, 3).hasClass("f-italic");
// If where the colModel ID is col2 has the "f-bold" class, it returns true, if else false.
grid.col("col2").hasClass("f-bold");

.removeClass(className)

Removes class of which is passed as a parameter, className, from the selected W5 Grid Parts element if the class exist.
The data type of the parameter is a String and can remove multiple classes by white space separator.

Example

// Remove "f-italic" class from 3rd row.
grid.row(2).removeClass("f-italic");
// Remove "f-bold" class from the column where the colModel ID is col2.
grid.col("col2").removeClass("f-bold");

.toggleClass(className, switch)

If the selected W5 Grid Parts element

does not have the className then perform
.addClass(className).
has className then perform
.removeClass(className).

The 2nd parameter switch

switch is optional parameter.
If the switch value is true, perform addClass(className) of className, if the value is false perform removeClass( className). If switch is designated, javacript code can be written as below.

var addOrRemove = grid.cell(0, 0).get("data");

// if addOrRemove is true, than add bgBlue class,
// or if it is false, remove bgBlue class.
grid1.col(1).toggleClass("bgBlue", addOrRemove);

This code is same as follows.

var addOrRemove = grid.cell(0, 0).get("data");

if ( addOrRemove ){
    grid.col(0).addClass("bgBlue");
} else {
    grid.col(0).removeClass("bgBlue");
}
