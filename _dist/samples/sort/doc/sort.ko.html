Sort

그리드 헤더를 더블클릭하거나 sort() API 를 이용하여 sroting 한다.

그리드 헤더

초기 상태에서 그리드 헤더를 더블클릭할 때마다 오름차순-내림차순-초기 상태를 순환합니다.
sort 를 원치않는 컬럼은 Column Model 에 sortable false 를 지정하여 방지합니다.

  grid1 = new w5.Grid({
    ....
    colModel : [{
      headerLable: "Column 1",
      sortable: false
    }]
  }).render();

sort API

sort() 메소드에 sort 할 컬럼과 방향을 전달하여 정렬시킵니다.
컬럼과 방향은 Array 로 다중 처리될 수 있습니다.
컬럼은 컬럼ID 또는 인덱스(시작은 0부터)를 지정합니다.
방향은 지정하지 않으면 오름차순 입니다.
파라미터가 전달되지 않으면 모든 정렬이 초기화됩니다.
사용자 comparator function 을 파라미터로 전달하면 해당 comparator 를 이용하여 정렬됩니다.

  // 'firstName' Column 으로 sort 합니다.
  // direction 기본값은 ascent 입니다.
  grid1.sort('firstName');

  // 파라미터를 전달하지 않으면 모든 sort 가 초기화됩니다.
  grid1.sort();

  // 첫 번째 컬럼('firstName') 을 descent 로 sort 합니다.
  grid1.sort( 0, 'desc' );

  // 'state' 컬럼과 두 번째 컬럼('lastName') 을 순차적으로 sort 합니다. 
  grid1.sort( ['state', 1] );

  // 'state', 'city', 첫 번째 컬럼('firstName')을 순차적으로 각각 'ascent', 'ascent', 'descent' 방향으로 정렬합니다.
  grid1.sort( ['state', 'city', 0], ['asc', 'asc', 'desc'] );

  // 사용자 comparator 를 작성하여 sort 할 수 있습니다.
  // city, companyName 컬럼을 순차적으로 정렬합니다.
  var comparator = function(item) {
    return [ item.get("city"), item.get("companyName") ];
  };
  grid1.sort(comparator);
