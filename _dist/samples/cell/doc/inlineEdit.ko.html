Inline Edit

Cell 의 Data 를 조작하는 방식은 displayType 이 text 인 경우와
selectbox, checkbox, radio 처럼 widget 의 아이템을 선택함으로써 value 가 결정되는 경우,
그리고 custom
이렇게 크게 3가지로 나뉜다.

Actionable Mode

Actionable Mode 시작

다음과 같은 동작을 통해 데이타를 수정하고 widget 을 수정할 수 있는 Actionable Mode 로 전환된다.

  • Cell DoubleClick
  • Cell Navigation Mode
    • Enter(Return)
    • F2
  • Tab(+ Shift) 으로 이동
    • Cell 선택과 함께 Actionable Mode 로 전환된다.

Actionable Mode 종료

다음과 같은 동작이 발생할 때 Actionable Mode 가 종료된다.

  • Blur 이벤트가 발생했을 때 - 그리드 외에 다른 영역을 클릭 등
  • Esc
  • Tab(+ Shift) 으로 다음/이전 Cell 로 이동

View - Model Binding

text

actionable Mode 가 종료되면 자동으로 View 와 Model 간 데이타가 동기화된다.

기본 Widget

selectbox, checkbox, radio 등은 마우스나 키보드로 선택한 아이템의 값이 actionable Mode 가 종료될 때 View 와 Model 간 데이타가 동기화된다.

custom - 자세한 내용은 Custom Cell 참조

각 Widget 에 정의된 이벤트를 통해 Model 에 데이타를 동기화시킨다.
