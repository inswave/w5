Data Type

w5는 number, string, boolean 총 3개의 Data Type 을 지원한다.
별도의 Data Type 을 지정하지 않으면 data 자체 type 을 그대로 유지한다.
Data Type 을 지정한 경우, Data 를 set 하거나 그리드 생성 시 지정한 Data Type 으로 cast 되어 Data 가 삽입됩니다.

Date

Date 객체는 w5 에서 지원하는 Primitive Data Type 이 아니기 때문에 그리드에 Set 하기 위해서는 지원하는 Type 으로 cast 해야 한다.
string type 으로 set 하면 w5 유료 버전에서 original format 과 format 을 이용하여 다양하게 표현할 수 있다.
예외적으로 displayType 이 custom 인 Cell 에 3rd Party Widget 등을 넣는 경우, 해당 Widget 에 Date 객체를 전달할 수 있다.

아래 예제는 총 30개의 row 를 가진 그리드이다.
col0, col1 모두 string type 의 데이타가 0부터 29까지 삽입된다.
col0 컬럼만 number dataType 을 설정한다.
col1 컬럼은 Default인 string dataType 이다.
컬럼 헤더를 더블클릭하여 sort 를 한다.
참고로 헤더를 더블클릭할 때마다 오름차순 정렬, 내림차순 정렬, 오리지널 순서 상태로 전환된다.
헤더의 아이콘으로 sort 상태를 구분할 수 있다.
sort 를 하기 위해서는 sortable 설정을 true 로 주어야 한다.

colModel[i].sortable = true;

col0 컬럼을 오름차순 소트 실행하면 dataType 이 number 이기 때문에 0, 1, 2, 3 ... 9, 10, 11 순서로 오리지널과 동일하다.
col1 컬럼을 오름차순 소트 실행하면 dataType 이 string 이기 때문에 0, 1, 10, 11 ... 19, 2, 21 순서로 정렬된다.
