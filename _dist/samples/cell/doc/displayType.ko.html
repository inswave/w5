Display Type

데이타를 화면에 표시하기 위한 유형으로 현재 지원하는 Display Type 은 다음과 같다.

  • text
  • select
  • checkbox
  • radio
  • link
  • img
  • button
  • custom

Template

text, custom type 은 template 을 이용할 수 있다.(Custom Cell 참조)

Event Handler

select, checkbox, radio type 은 change 이벤트에 대한 callback handler 를 지정할 수 있다.
Column Meta 객체의 change attribute 에 handler function 을 지정한다.(직접 해보세요! 02을 참고하세요.)

Example

아래는 col2 컬럼의 select 값이 변경되었을 때 col3 컬럼의 select options 를 동적으로 필터링해서 재설정하는 예제이다.

columnObject.callback = {
    change : function( value, pos ) {
        this.cell( pos.row, 'col3' ).set( 'options', filterOptions(value) );
    }
}

Callback Handler Context 및 파라미터

callback handler 함수 내 this 는 그리드 자체를 참조한다.
callback handler 에 전달되는 파라미터는 다음과 같다.

  • value : 이벤트가 발생한 Cell 의 값
  • pos[Object] : 이벤트가 발생한 Cell 의 좌표

pos 파라미터(Object)의 Attributes 는 다음과 같다.

row col
이벤트가 발생한 row 좌표 이벤트가 발생한 column 좌표

일반

그리드 생성 시 데이타 등과 함께 컬럼에 대한 Meta 정보를 정의하도록 되어 있다.

조건에 따른 displayType 설정 - expression 활용

Cell 이 포함된 Row 의 다른 컬럼 값에 따라 해당 Cell 의 displayType 을 expression 을 이용해서 지정할 수 있다.

// obj 는 특정 컬럼의 meta 정보를 담는 객체
obj.expression = function ( options ) {
    // 첫 번째 컬럼(col0)의 값을 얻어 온다.
    displayType = options.attributes['col0'];

    // 첫 번째 컬럼의 값에 따라 두 번째 컬럼의 displayType 를 지정한다.
    // this 는 그리드 자체를 참조한다.
    // silent 옵션을 true 로 주어 화면 갱신이 중복으로 발생하는 것을 방지한다.
    this.cell( options.rowIndex, options.colId ).set( 'displayType', displayType, { silent: true } );

    return function (options) {
        // row 전체 data(JSON - options.attributes) 에서 해당 컬럼의 데이타를 반환한다.
        return options.attributes[options.colId];
    }
};

expression function 에 전달되는 파라미터(options) 는 Object 로 다음과 같은 정보를 담고 있다.

attribute value
rowIndex row 의 index
colId column 의 ID
attributes 해당 Row 의 데이타를 담은 JSON 객체

attributes JSON 객체는 컬럼 ID 를 attribute 명으로 해당 컬럼에 해당하는 Cell 의 데이타를 value 로 담고 있다.
아래는 attributes JSON 객체에 대한 예시다.

attribute value
col0 'radio'
col1 '3'
col2 '0'
col3 '2'
col4 '0'
col5 '3'
col6 '07/08/2014'
col7 Mon Jun 30 2014

Real-Time

w5 에서 제공하는 Cell, Column 에 접근하는 방식과 set function 을 통해 실시간으로 displayType 을 변경할 수 있다.(직접 해보세요! 01을 참고하세요.)

// It changes the cell(0,0) displayType to 'button' from 'text'
grid1.cell(0,0).set('displayType', 'button');

// It changes the cell(0,1) displayType to 'radio' from 'select'    
grid1.cell(0,1).set('displayType', 'radio');

// It changes the 4th column displayType to 'radio' from 'select'    
grid1.col(3).set('displayType', 'select');
