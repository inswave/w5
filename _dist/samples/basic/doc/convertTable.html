Converting a table to a W5 grid

As mentioned before, a HTML table can be converted to a W5 grid. Besides the table data, the style specified for the table can also be converted without changes.

var grid1 = new w5.Grid({
    el : "#tblMain",
    parseTable : true,
    option: {
      width : "650px",
      height : "400px",
      caption : "convert to Grid",
      rowNum : 10,
      vScrollDegree : 5
    }
  }).render();

Unlike a general method of creating a grid, when converting the table, the table id should be given to the el property and the parseTable property should be set to true. When the W5 grid parses a table, it sets the <td> or <th> of the <table> <thead>to headerLabel. If the <thead>is not available, the first <tr> is assigned to headerLabel.
headerLabel can be overridden by colModel.
Other colModel definition and default value designation are same.
