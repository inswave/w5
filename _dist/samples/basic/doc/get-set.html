.get()/.set() API

The basic manipulation can be done by .get()/.set() API from W5 Grid.

API Structure

W5 Grid API structure looks like as below:

grid.col("name").get("data");

col()
Before calling get(), set() API, select cell such as cell, row, col, table by method.
This stage is called W5 Grid parts select method.
get()
Pull the specific Data or Property by get() or set by set()

Here are the examples.

// Pull the data of the 4th row of which key is price. And set its cell style(font-weight) to bold.
grid.cell(3, "price").set("style", "font-weight" "bold");

// Pull the data of which key is name. And set its width to 200px.
grid.col("name").set("width", "200px");

// Pull the data of the 6th row and set its background to red.
grid.row(5).set("style", "background", "red");

// Set all the data to -.
grid.table().set("data", "-");

W5 Grid parts select method

Grid parts is the components of the W5 Grid. These components are composed with cell, col, row, and table. W5 Grid provides select method for the ease of handling. select method:

  • index starts from 0.
  • cell( row index, col index | colModel id)
  • row( row index )
  • col( col index | colModel id )
  • table()

col, row, and table are the assembly of designated area. Therefore, it also can be expressed as cell(row, col) method.
The parameter of * selects all the cells, cell(" * ", col index)is same as a row, cell(row index, " * ") is same as a col. Also * as both parameter, it is same as selecting the entire table.

grid.cell(8, "*")...;      // Same as grid.row(8);
grid.cell("*", 3)...;      // Same as grid.col(3);
grid.cell("*", "*")...;    // Same as grid.table();

.option() method

In order to manipulate(get(), set()) Grid setting such as colOrder, frozen column, you need to approach through option() method.

grid.option().set("frozenColumn", 3);    // Set the column to freeze until 3rd row.

The first property parameter is explained in Create W5 Grid option.

Chaining

W5 Grid supports Chaining. Select the corresponding element by W5 Grid Parts select method and can continuously use set(). get() returns a value and ends the Chaining.
Here is the example of Chaining.

grid.col("price")                        // pull data of which id is "price"
    .set("style", "background", "red")  // and sets its background to "red"
    .set("displayType", "text")         // and sets its displayType to "text"
    .set("class", "font-white");        // and sets its css class to "font-white".

.get()

.get() is used to get the property of the corresponding Grid parts.

data

The selected Grid parts data can be obtained by as follows:

// Pull the data of column of which key is "name".
grid.col("name").get("data");

The returned value's type is determined as Grid Parts select Method.

W5 Grid Parts select method Return type
Case of cell string
Case of col Array
Case of row
default
Object
option type : "array"
Array
Case of table Object Array

Refer to the Process Data for more information on get().

option { type : "array" }

If array format is needed instead of object when grid parts is a row

// Pull 2nd row data in array format.
grid.row(1).get("data", { type : "array" });

change the last parameter as object format and give type as an 'array`.

Preview from browser Console

Style

Obtains the style information from selected W5 Grid parts. There are 2 ways to obtain Style Property by .get().

// Obtain all the style information of which colModel ID is "name".
grid.col("name").get("style");                    
// Obtain a background value of which colModel ID is "name".
grid.col("name").get("style", "background");

As the first method, .get("style"), it obtains all the style object applied to selected W5 Grid parts and returns in form of object. As the second method, .get("style", 'property'), it obtains the style of the 'property' and returns string.

class

Obtains the CSS Class applied to the selected W5 Grid parts.

// Obtain the CSS Class of which colModel ID is "name".
grid.col("name").get("class");

return value is a stirng, and if several classes are applied it separates by the white space.

.set()

.set() is used to set value to property of the selected Grid parts.

data

Set data value to the selected Grid parts..

var data = [1,2,3,4,5,6,7];
grid.col("name").set("data", data );

Data type that can be set to each Grid parts are as follows:

W5 Grid Parts select method Data type
Case of cell string
Case of col Array
Case of row Object | Array
Case of table Object Array

Style

The API to set the style of the selected W5 Grid parts is as follows:

grid.col("name").set("style", property, value );
grid.col("name").set("style", object);

W5 Grid manages the Style in Object form, so as the first example, style can be set by key, or as the second example, style can be set by whole.
W5 Grid Style is easy to decorate. If a specific cell property has been set by several times, the last set style will be applied. For more information, please refer to Style concept.

class

CSS class can be set/get to the selected W5 Grid parts.

grid.col("name").get("class", string);

If more than one CSS Classes to be set, give it a white space to separate from each other.

grid.col(2).set("class", "bg-yellow f-bold");

option { silent : true | false }

Decides to display. if the value is true, .set() properties will not display. the default value is false.

// Set entire grid to "bold" but does not display.
grid.table().set("style", "font-weight", "bold", { silent : true });
