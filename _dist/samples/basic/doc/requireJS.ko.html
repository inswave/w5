RequireJS

RequireJS 를 이용해서 w5 와 종속성이 있는 모듈들을 관리하고 이용할 수 있다.
w5 는 jQuery, Undersocre.js, Backbone.js 모듈에 종속성이 있다.

Examples:

1 requireJS 를 import 하는 script tag 의 data-main attribute 를 이용해서 configuration 을 설정한다.

<script src="js/lib/require.js" data-main="js/app/requirejs.main"></script>

2 data-main Entry Point 에 사용된 js 파일에 다음과 유사하게 설정한다.

requirejs.config({
  baseUrl: 'js/lib/',
  paths:{
    jquery:'1.10.2/jquery.min',
    underscore: '1.6.0/underscore-min',
    Backbone: '1.1.2/backbone-min',
    w5: '0.6.0/w5.min'
  },
  shim: {
    w5: ['jquery', 'underscore', 'Backbone']
  }
});

3 export 된 모듈들을 이용한다.

requirejs( ['jquery', 'w5'], function( $, w5 ) {
  $("body").append("<div id='grid1'>");
  grid1 = new w5.Grid({
    el : "#grid1",
    생략....
  }).render();
});
