RequireJS

W5 and dependent modules can be managed and utilized, using RequireJS.
W5 is dependent upon the jQuery, Undersocre.js, Backbone.js module.

Examples:

1 Set the configuration using data-main attribute of the scrip tag that imports RequireJS.

<script src="js/lib/require.js" data-main="js/app/requirejs.main"></script>

2 Set as described below in the JS file used for the data-main entry point.

requirejs.config({
  baseUrl: 'js/lib/',
  paths:{
    jquery:'1.10.2/jquery.min',
    underscore: '1.6.0/underscore-min',
    Backbone: '1.1.2/backbone-min',
    w5: '0.6.0/w5.min'
  },
  shim: {
    w5: ['jquery', 'underscore', 'Backbone']
  }
});

3 Use the exported module.

requirejs( ['jquery', 'w5'], function( $, w5 ) {
  $("body").append("<div id='grid1'>");
  grid1 = new w5.Grid({
    el : "#grid1",
    Omitted....
  }).render();
});
