Defaults

When a row is created with defaults, default value is applied to the column where no value exist.
3 methods can be used as follows:

Column Model

Set the default value which will be used for the column in question, to colModel attribute such as width, and header.

// Set default value to  firstName, lastName, companyName.
var colModel = [{ "width": 80, "headerLabel": "firstName", "default": "shye", "style": "font-weight: bold" },
                { "width": 80, "headerLabel": "lastName", "default": "lee" },
                { "width": 140, "headerLabel": "companyName", "default": "inswave" },
                { "width": 150, "headerLabel": "address" },
                ....];

// colModel setting is same as general method.              
var grid1 = new w5.Grid({
    ....
    colModel : colModel
}).render();

// Prepare the data to input.
// No data exist for firstName, lastName, companyName.
var tempData = { "address": "6 Greenleaf Ave", "city": "San Jose", "county": "Santa Clara", "state": "CA", "zip": "95111", "phone1": "408-540-1785", "phone2": "408-813-4592", "email": "vinouye@aol.com", "web": "http://www.cnetworkinc.com" };

// Insert a row in 2nd position.
// The omitted column data will be set with default value.
grid1.addRow( 1, tempData );

setDefaults / getDefaults

Set default value dynamically with setDefaults() method. Set can be done after a grid has been created.
All instance(Model) is being shared, because the Object is being conveyed by reference. Recommend to use Backbone function to set defaults.

    defaults = function () {
      return {
        firstName: 'youngsoo',
        lastName: 'choi',
        companyName: 'inswave systems'
      };
    };

    // Pass the functions by defaults.
    // The default value of Object returned from the function will be applied to a Row when added.
    grid1.setDefaults( defaults );

Backbone.Model

Set defaults with Backbone.Model.
When defaults is being set due to the higher priority than Column Model or setDefaults method, the value of which extend set to Model is applied.

    // Set defaults to Model extend
    model = Backbone.Model.extend( { defaults: function() {
      return {
        firstName: 'A',
        lastName: 'B',
        companyName: 'C'
      };
    } } ),

    // At the point of Collection creation, set above Model.
    collection = new Backbone.Collection( JSON.parse(data), { model: model } ),
