Process Data

Data can be manipulated by Cell, Row, Column, Table unit.
See get/set Guide.

READ

Select the target with parts selector and obtain the data by get() method.

// Returns the data of the first row of the first cell.
// James
grid1.cell(0,0).get('data');

// Returns the entire 6th row data in form of an Object.
// {"firstName":"Mitsue", "lastName":"Tollner" .... , "web":"http://www.morlongassociates.com"}
grid1.row(5).get('data');

// Can manipulate the data by direct reference to Backbone.Collection.
// Returns the same result as above 2 commands.
grid1.collection.at(5).toJSON();

// Returns the same data is above but exclude column ID and returns as an Array form.
// ["Mitsue", "Tollner" .... , "http://www.morlongassociates.com"]
grid1.row(5).get('data', { type: 'array' });

// Only returns column data that has "pick" as an option.
// {"firstName":"Mitsue", "companyName":"....", "city":"....", "phone1":"....", "email":"mitsue_tollner@yahoo.com"}
grid1.row(5).get('data', { pick: ['firstName', 'companyName', 'city', 'phone1', 'email'] });

// Returns column data except the colums with "opmit" option.
// The result is same as above command.
grid1.row(5).get('data', { omit: ['lastName', 'address', 'county', 'state', 'zip', 'phone2', 'web'] });

// Returns data regarding city column in form of an Array.
// The Array length is same as Row number.
// ["New Orleans", "Bridgeport", ".... , "Rockford", "Aston"]
grid1.col('city').get('data');

UPDATE

Select method is same as READ, and data can be changed by set() method. For validation, check out the Pro version . 

// Change the first row of the companyName to `Beck Horizon Builders`, and change its text color to `OrangeRed`.
grid1.cell( 0, 'companyName' ).set( "data", "Beck Horizon Builders" ).set( "style", "color", "OrangeRed" );

CREATE

Add a new Row by addRow() method.
The followings are the 2 parameters to set.

  • Index
    • If the Index is not set, it goes to the last.
    • To insert the Index in the first, insert 0.
  • Data to be Inserted

The supported data Format is as follows:

  • Object
  • Array(Only Data)
  • JSON Array
  • Two-Dimensional Array(Only Data)

For more information on defualt value, see Defaults. 

  // The total Row number can be obtained by getRowLength() method.
  // The sample has total of 20 rows.
  grid1.getRowLength();

  // If no parameter is set, an empty Row is added to the last.
  // You can see a Row is added by scrolling.
  grid1.addRow();

  // With getRowlength() method, it returns 21 as total row number.
  grid1.getRowLength();

  // Set the position of where the number type Row gets added.
  // An empty row is added to the 2nd row.
  grid1.addRow(1);

  // You can set the position and the data in order.
  // Multiple Rows can be added by Two-Dimensional Array or JSON Array.
  // Row added to 2nd and 3rd row as `Veronika` and `Willard`.
  grid1.addRow( 1, [[ "Veronika", "Inouye", "C 4 Network Inc", "6 Greenleaf Ave", "San Jose", "Santa Clara", "CA", "95111", "408-540-1785", "408-813-4592", "vinouye@aol.com", "http://www.cnetworkinc.com"],
[ "Willard", "Kolmetz", "Ingalls, Donald R Esq", "618 W Yakima Ave", "Irving", "Dallas", "TX", "75062", "972-303-9197", "972-896-4882", "willard@hotmail.com", "http://www.ingallsdonaldresq.com"]] );

  // If the parameter has only data without position, the row gets added to the last.
  // Data inserted to the column where the property of the Object and the Column ID is identical.
  grid1.addRow( { "firstName": "Maryann", "lastName": "Royster", "companyName": "Franklin, Peter L Esq", "address": "74 S Westgate St", "city": "Albany", "county": "Albany", "state": "NY", "zip": "12204", "phone1": "518-966-7987", "phone2": "518-448-8982", "email": "mroyster@royster.com", "web": "http://www.franklinpeterlesq.com" } );

  // If the Index is 0, a Row inserted at the first.
  // Array does not have property, so the Data inserted to the column in order.
  // If the number of the Array data is greater than the number of the column, the left over Array data is ignored.
  // If the number of the column is greater than the Array data, the rest column stays empty.
  grid1.addRow( 0, [ "Alisha", "Slusarski", "Wtlz Power 107 Fm", "3273 State St", "Middlesex", "Middlesex", "NJ", "8846", "732-658-3154", "732-635-3453", "alisha@slusarski.com", "http://www.wtlzpowerfm.com" ] );

DELETE

Row can be deleted by removeRow() method.
Obtain deleted row by removedRow() method.
The row added by addRow() method is excluded, and only includes the very first rows when a grid is created (by fetch). 

  // Delete 2nd row.
  grid1.removeRow( 1 );

  // Obtain deleted row.
  // The return value is an Array.
  // If deleted row exist, it stays in Object form in Array Item.
  // [{"firstName":"James","lastName":"Butt","companyName":"Beck Horizon Builders","address":"6649 N Blue Gum St","city":"New Orleans","county":"Orleans","state":"LA","zip":"70116","phone1":"504-621-8927","phone2":"504-845-1427","email":"jbutt@gmail.com","web":"http://www.bentonjohnbjr.com"}]
  grid1.removedRow();

Give it a try!

If fetch is set when a grid is created, data will be obtained by the fetch and draw a grid.
For more information see fetch from Backbone.Collection.

var grid1 = new w5.Grid({
    .... ,
    fetch: {
      success: function() {
        ....
      }
    }
}).render();
