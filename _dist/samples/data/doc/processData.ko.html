Process Data

Cell, Row, Column, Table 단위로 데이타를 조작할 수 있습니다.
get/set 가이드를 참조하세요.

READ

parts selector 로 대상을 선택 후 get() 메소드를 이용해 데이타를 얻습니다.

// 첫 번째 Row 의 첫 번째 Cell 에 대한 Data를 반환합니다.
// James
grid1.cell(0,0).get('data');

// 여섯 번째 Row 전체 의 데이타를 Object 로 반환합니다.
// {"firstName":"Mitsue", "lastName":"Tollner" .... , "web":"http://www.morlongassociates.com"}
grid1.row(5).get('data');

// Backbone.Collection 을 직접 참조하여 데이타를 조작할 수 있습니다.
// 바로 위 2개의 명령과 동일한 결과를 반환합니다.
grid1.collection.at(5).toJSON();

// 위의 명령들과 동일한 데이타를 반환하지만 column ID 는 제외한 data 들만으로 구성된 Array 를 반환합니다.
// ["Mitsue", "Tollner" .... , "http://www.morlongassociates.com"]
grid1.row(5).get('data', { type: 'array' });

// 컬럼 중 pick 옵션에 지정한 컬럼들의 데이타만 반환합니다.
// {"firstName":"Mitsue", "companyName":"....", "city":"....", "phone1":"....", "email":"mitsue_tollner@yahoo.com"}
grid1.row(5).get('data', { pick: ['firstName', 'companyName', 'city', 'phone1', 'email'] });

// 컬럼 중 omit 옵션에 지정한 컬럼들을 제외한 데이타만 반환합니다.
// 결과는 바로 위 명령과 동일합니다.
grid1.row(5).get('data', { omit: ['lastName', 'address', 'county', 'state', 'zip', 'phone2', 'web'] });

// city 컬럼에 대한 데이타만으로 이뤄진 Array로 반환합니다.
// Array length 는 Row 갯수와 동일합니다.
// ["New Orleans", "Bridgeport", ".... , "Rockford", "Aston"]
grid1.col('city').get('data');

UPDATE

대상 선택은 READ 와 동일하며 set() 메소드를 이용해 데이타를 변경합니다.
validation 은 유료 버전을 참고하세요.
서버와의 동기화는 여기를 참조하세요.

// 첫 번째 Row 의 companyName 을 'Beck Horizon Builders' 로 수정하고 글자색을 'OrangeRed' 로 변경합니다.
grid1.cell( 0, 'companyName' ).set( "data", "Beck Horizon Builders" ).set( "style", "color", "OrangeRed" );

CREATE

addRow() 메소드를 이용하여 Row 를 추가합니다.
파라미터로 다음 2가지를 지정할 수 있습니다.

  • Index
    • 지정하지 않으면 맨 마지막에 삽입된다.
    • 맨 처음에 삽입하기 위해서는 0 을 삽입한다.
  • Insert 될 데이타

지원하는 데이타 형태는 다음과 같습니다.

  • Object
  • Array(Only Data)
  • JSON Array
  • 2차원 Array(Only Data)

defualt value 는 Defaults를 참조하세요.
서버와의 동기화는 여기를 참조하세요.

  // Row 의 총 갯수를 getRowLength() 메소드를 이용해서 가져올 수 있습니다.
  // 샘플 최초의 갯수는 20 입니다.
  grid1.getRowLength();

  // 파라미터가 아무 것도 없다면 그리드 맨 마지막에 빈 Row 가 삽입됩니다.
  // 스크롤을 통해 맨 마직에 Row 가 추가된 것을 확인할 수 있습니다.
  grid1.addRow();

  // 다시 getRowLength() 메소드를 이용해서 Row 갯수를 가져오면 21 임을 확인할 수 있습니다.
  grid1.getRowLength();

  // number type 으로 Row 가 삽입될 위치를 지정합니다.
  // 2번째 Row 에 빈 Row 가 삽입됩니다.
  grid1.addRow(1);

  // 삽입될 위치와 데이타를 순서대로 지정할 수 있습니다.
  // 2차원 배열이나 JSON Array 형태로 데이타를 지정하면 한 번에 여러 개의 Row 를 삽입할 수 있습니다.
  // 2번째와 3번째 Row 에 각각 `Veronika` 와 `Willard` 로 시작하는 Row 가 삽입된다.
  grid1.addRow( 1, [[ "Veronika", "Inouye", "C 4 Network Inc", "6 Greenleaf Ave", "San Jose", "Santa Clara", "CA", "95111", "408-540-1785", "408-813-4592", "vinouye@aol.com", "http://www.cnetworkinc.com"],
[ "Willard", "Kolmetz", "Ingalls, Donald R Esq", "618 W Yakima Ave", "Irving", "Dallas", "TX", "75062", "972-303-9197", "972-896-4882", "willard@hotmail.com", "http://www.ingallsdonaldresq.com"]] );

  // 위치 지정없이 데이타만 파라미터로 지정하면 맨 마지막에 Row 가 생성됩니다.
  // Object 의 property 와 그리드의 Column ID 가 일치하는 컬럼에 데이타가 각각 삽입됩니다.
  grid1.addRow( { "firstName": "Maryann", "lastName": "Royster", "companyName": "Franklin, Peter L Esq", "address": "74 S Westgate St", "city": "Albany", "county": "Albany", "state": "NY", "zip": "12204", "phone1": "518-966-7987", "phone2": "518-448-8982", "email": "mroyster@royster.com", "web": "http://www.franklinpeterlesq.com" } );

  // 인덱스를 0 을 주면 첫 번째 Row 로 삽입됩니다.
  // Array 는 별도의 property 가 존재하지 않음으로 데이타 순서대로 컬럼에 데이타가 삽입됩니다.
  // 컬럼 갯수보다 데이타 숫자가 많은 경우, 넘치는 데이타는 무시됩니다.
  // 컬럼 갯수보다 데이타 숫자가 적을 경우, 앞 컬럼부터 데이타를 채우고 나머지는 빈 Cell 로 표시됩니다.
  grid1.addRow( 0, [ "Alisha", "Slusarski", "Wtlz Power 107 Fm", "3273 State St", "Middlesex", "Middlesex", "NJ", "8846", "732-658-3154", "732-635-3453", "alisha@slusarski.com", "http://www.wtlzpowerfm.com" ] );

DELETE

removeRow() 메소드를 이용하여 Row 를 삭제합니다.
removedRow() 메소드를 이용해서 삭제된 Row 를 얻어 옵니다.
addRow() 메소드를 통해 추가된 Row 는 제외되며 (fetch 로) 그리드 생성 시점의 최초 Row 만 해당됩니다.
서버와의 동기화는 여기를 참조하세요.

  // 2번째 Row 를 삭제합니다.
  grid1.removeRow( 1 );

  // 삭제된 Row 를 얻어 옵니다.
  // return 값은 Array 입니다.
  // 삭제된 Row 가 있는 경우, Array Item 에 Object 로 존재합니다.
  // [{"firstName":"James","lastName":"Butt","companyName":"Beck Horizon Builders","address":"6649 N Blue Gum St","city":"New Orleans","county":"Orleans","state":"LA","zip":"70116","phone1":"504-621-8927","phone2":"504-845-1427","email":"jbutt@gmail.com","web":"http://www.bentonjohnbjr.com"}]
  grid1.removedRow();

직접 해보세요!

그리드 생성 시점에 fetch 를 설정하면 생성과 동시에 fecth 를 통해 데이타를 얻어서 그리드를 그립니다.
자세한 내용은 Backbone.Collection 의 fetch 을 참조하세요.

var grid1 = new w5.Grid({
    .... ,
    fetch: {
      success: function() {
        ....
      }
    }
}).render();

단, 아래 샘플의 경우에는 샘플 소스와 데이타를 얻어오는 도메인이 달라서 발생하는 'Access-Control-Allow-Origin' 이슈를 처리하기 위해 jsonp 를 이용합니다.
jsonp callback function 으로 success 를 사용하고 있지만, 일반적으로는 필요없는 부분입니다.
