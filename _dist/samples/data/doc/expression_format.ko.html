Expression

특정 Cell 에 대한 데이타나 메타 정보를 해당 Cell 이 포함된 Row 의 다른 Cell들의 데이타와 메타 정보를 이용하여 동적으로 생성하는 기능을 제공한다.
Custom Cell 과 Display Type 을 참조하세요.

Format

Column Model 에 Format function 을 지정하여 원래 data 를 지정한 형식으로 표현한다.
Format function 에 전달되는 파라미터는 원래 data 이다.
다양한 Number/Date Formet 은 유료버전을 확인하세요.

  var grid1 = new w5.Grid({
    ....
    colModel : [{ ....
                  // format attribute 에 function 을 지정한다.
                  "format": function(data) {
                    return data.substring(0, 4) + '/' + data.substring(4, 6) + '/' + data.substring(6);
                  }
                },
                { ....
                  "format": function(data) {
                    return Number( ( data * 100 ).toFixed(2) );
                  }
                }]
  }).render();

직접 해보세요!

Cell 을 더블클릭하거나 Enter 또는 F2 Key 를 입력하여 Edit 모드로 전환되면 원래 데이타가 보여지며 수정할 수 있다.
