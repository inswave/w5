Flexible Column

Flexible Column은 해당 Column의 width를 전체 그리드 너비의 비율로 정의합니다. 비율로 정의하기 때문에 가로 스크롤을 생성하지 않고 그리드의 너비가 변하면 정의된 flex 정수 만큼 각 column들의 너비가 자동 조정 됩니다. 

var colModel = [
        { "flex" : 1, "id" : "col1" },
        { "flex" : 2, "id" : "col2" },
        { "flex" : 3, "id" : "col3" },
        { "flex" : 4, "id" : "col4" }
    ];
var grid = new w5.Grid({
        ...
        option : {
            width: "500px"
        },
        colModel : colModel,
        ...
    });

위와 같이 colModel(Column Model)에 flex를 정의하고 width가 500px인 W5 Grid를 생성하면, 첫번째 Column인 "col1"의 column의 width 50px이 되며 "col2"의 width는 100px, "col3"의 width는 150px, "col4"의 witdh는 200px이 됩니다.

생성 이후 이 그리드의 width를 조정하면 Column들이 자동 조정되어 각각 column의 width가 100px, 200px, 300px, 400px이 되는 것을 확인 할 수 있습니다.

grid.option().set("width","1000px");

grid.col(0).get("width"); // return 100
grid.col(1).get("width"); // return 200
grid.col(2).get("width"); // return 300
grid.col(3).get("width"); // return 400

Flexible Column은 W5 그리드 생성 이후에도 .set()/get() API로도 지정이 가능합니다.
Flex로 지정됐던 Column의 width가 고정단위(px)로 바뀌면, 그리드의 전체 크기에서 고정단위의 width를 뺀 나머지에 대해 Flex Column들을 재조정합니다.

grid.option().set("width","500px");
grid1.col(1).set("width", 100);
grid1.col(2).set("width", 100);

grid1.col(0).get("width"); // return 60
grid1.col(1).get("width"); // return 100
grid1.col(2).get("width"); // return 100
grid1.col(3).get("width"); // return 240
