Column 이동하기

Column 이동도 API로 이동하는 방법과, UI로 조작하는 방법이 있습니다.

grid.moveColumn(옮겨질 컬럼 인덱스, 옮길 컬럼 인덱스)

Column 이동하기 API 사용법은 매우 간단합니다. 옮겨질 컬럼 인덱스와 옮길 컬럼 인덱스를 인자에 차례대로 넣으면 됩니다.
예를들어 2번째 Column 을 5번째 Column 으로 이동시키고 싶다면,

grid.moveToColumn(2, 5);

라고 javascript code를 작성하면 됩니다.

UI로 Column 이동하기

UI로 Column 옮기기 위해서는 옮겨질 컬럼의 라벨을 드래그해서 원하는 컬럼의 헤더에 드롭하면 됩니다.
드롭 할 경우 적절치 않은 위치이면 마우스 커서의 모양이 "not-allowed"로 변하고, 그렇지 않고 올바른 경우 "copy"로 변합니다.
커서 모양의 경우 css3의 추가된 cusor를 따르며, 브라우저의 지원 여부 혹은 브라우저 종류 마다 다를 수 있습니다.
