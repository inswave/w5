Column Width 조정하기

Column Width를 조정하기 위한 방법은, W5 Grid parts select method를 사용한 .set() API를 사용하는 방법과, UI로 조정하는 방법이 있습니다.

grid.col( index | id ).set("width", 세팅할 column의 width px )

argument를

  • index로 했을 경우

    0 부터 시작하며, 보이는 컬럼의 순서만 고려합니다.
    데이터 원본의 index로 접근하고 싶다면 _col(index) select method 사용하세요.

  • id로 했을 경우

    colModel의 id로 column의 unique 값으로 셀렉트하며, 이는 index와 달리 보이지 않는 컬럼도 처리가 가능합니다.

grid란 변수에 W5 Grid를 생성했다고 가정하면, 조작하고 싶은 column의 인덱스를 .col() 셀렉터로 지정하여 set API로 지정할 width값을 줍니다.

var grid = new w5.Grid({ ... }).render();
...
grid.col("name").set("width", "80px"); // Column id가 "name"인 Column의 width를 80px로 지정합니다.
grid.col(2).set("width", "250px"); // 세번째에 해당하는 Column의 width를 250px로 지정합니다.

width로 지정하는 width 값은 string이 아닌 number도 가능합니다.

UI로 Column width 조정하기

column label에 마우스를 column 경계선에 오버하면 파란색 바가 나타나고 마우스 포인터가 col-resize로 변합니다. 이때 원하는 만큼 드래그 앤 드롭하면 드롭한 영역까지 column의 width가 조정됩니다.
W5 Grid의 영역을 벗어나도 드래그할 수 있어 간격 조정에 용이합니다.
