Adjusting column width

To adjust column width, use .set() API by W5 Grid parts select method or use UI adjustment method.

grid.col( index | id ).set("width", Width px of the column to set )

If the augment is:

  • index

    It begins with 0, and only considers the visible columns order. If you want to access using the index of the data source, use _col(index) select method.

  • id

    It selects the unique value of the column by the id of the colModel. Unlike index, the hidden column can also be processed.

    Assume W5 Grid is created in a variable called "grid", specify the index of the column to manipulate with the .col() and assign a value of width, using the set API

var grid = new w5.Grid({ ... }).render();
...
grid.col("name").set("width", "80px"); // Specify the width of the column of which ID is `name` to 80px.
grid.col(2).set("width", "250px"); // Specify the width of the column corresponding to the third one to 250px.

The width value can have a number value, instead of a string value.

Adjusting column width using the UI

Position the mouse pointer over the column boundary of the column label, a blue bar will appear and the mouse pointer will be changed to col re-size state. Click and hold the boundary and move as much as intended. Release the mouse button to set the column width. It is easy to adjust a space, as the mouse pointer can be dragged out of the W5 grid area.
