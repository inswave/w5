Keyboard

There are broadly two modes ? navigation mode to move among cells, and actionable mode to manipulate the cell data and widget.

Navigation Mode

  • Right/Left Arrow Key
    Column movement. Row change is not allowed.

  • Up/Down Arrow Key
    Row movement. Column change is not allowed.

  • Home Key
    Moves to the first cell of the current row

  • End Key
    Moves to the last cell of the current row

  • Page Up Key
    Moves to the first cell of the current column

  • Page Down Key
    Moves to the last cell of the current column

Actionable Mode

Tab Key

  • Moving to the next actionable item.
    • An actionable item refers to an element that takes focus, switches to an editing state, or can be manipulated.
  • Switching to actionable mode when moved.
    • The editing mode will be engaged if the DisplayType is text.
    • If there is a widget in the cell, the widget will be focused.
      • Focused by the first child.
      • You can designate a selector to access the focusing object to the focusSelector property of the column model object, if the cell is a custom cell and the first child is not a tabbable object.
  • If there is a tabbable object (checkbox) in the cell, use the tab to move between items (checkboxes).
    • Focus moves to the next cell at the last tabbable item.
  • At the last column, the focus moves to the first column of the next row.
  • Shift + Tab operates contrary to the Tab.

focusSelector

  • Widget structure
<td>
  <span class="widget datetimepicker">
    <span class="picker-wrap">
      <input type="text" data-role="datetimepicker" class="input">
    </span>
  </span>
</td>
  • focusSelector designation
  grid1 = new w5.Grid( {
    Omitted...
    colModel: [{
        displayType: "custom",
        template = "<input type='text'/>",
        focusSelector = "input"
    }]
  });

Enter, F2 Key

Switching to actionable mode.

Escape Key

Switching from actionable mode to navigation mode.

Example

  • checkbox
    • Supports movement among items using Tab / Shift + Tab, like native action.
  • selectbox
    • Supports movement among items using the up/down arrow key, like native action.
  • radio
    • Supports movement among items using the left/right (up/down) arrow key, like native action.
  • custom
    • Among jQuery UI and Kendo UI datepicker that are inserted as a 3rd party widget, the calendar pop-up of the Kendo UI can be populated using alt (option) + down arrow key.
