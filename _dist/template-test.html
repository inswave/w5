Data 조작

Data 값 설정

.set("data", value) .set() api를 이용하여 data 값을 설정할 수 있습니다. .set()으로 data를 설정할 경우 어떤 요소에 적용할 것 인지에 따라 값이 설정되는 방법이 달라질 수 있으므로 주의가 필요합니다.

  1. cell에 적용 할 때에는 value를 해당 cell의 data로 설정합니다.
  2. row나 column에 적용 할 때에는 value를 1차원 배열로 간주합니다. 이때 value에 담겨진 값 들은 차례대로 row나 column에 속한 각각의 cell의 data 값으로 설정됩니다. value의
  3. table에 적용 할 때에는 value를 2차원 배열로 간주합니다. 이때 value에는 row의 data에 해당하는 1차원 배열이 담겨있습니다. 이 배열들이 차례대로 row의 data로 설정됩니다.
grid.cell(0, 0).set("data", "a");         // cell의 data 값을 "a"로 설정
grid.col(0).set("data", ["a", "b", "c"]); // column의 data 값을 설정. 3개 cell의 값이 바뀐다
grid.set("data", [["a", "b", "c"], ["d", "e", "f"]]); // table의 data 값을 설정. 2개 row의 값이 바뀐다

Data 값 읽기

.get("data") .get api를 이용하여 selector로 선택된 첫 번째 요소의 data 값을 구할 수 있습니다. .set()과 마찬가지로 어떤 요소에 적용되느냐에 따라 다른 형태의 값을 반환합니다.

  1. cell에 적용할 경우 단일 cell의 data 값을 반환합니다.
  2. row나 column에 적용할 경우 전체 row나 column의 data 값을 1차원 배열로 반환합니다.
  3. table에 적용할 경우 전체 cell의 data 값을 2차원 배열로 반환합니다.
var value = grid.cell(0, 0).get("data"); // cell의 data를 반환
var array = grid.row(0).get("data");     // row의 data를 1차원 배열로 반환
var matrix = grid.get("data");           // grid 전체 data를 2차원 배열로 반환

Style 조작

Style 값 설정

.set("style", [key], value)
.set() api를 이용하여 style 값을 설정할 수 있습니다. .set()을 이용하여 style 값을 설정하면 각각의 style 요소 별로 가장 마지막에 설정한 값이 적용됩니다.
key가 주어진다면 style의 특정 요소의 값을 설정할 수 있으며, 이때 value는 string 값으로 간주합니다. key가 주어지지 않는다면 value는 설정할 여러 가지 style 값들을 나타내는 object로 간주합니다.

// row의 배경색은 검정, 글자색은 파랑으로 설정
grid.row(0).set("style", {
  background: "black",
  color: "blue"
});
// column의 배경색을 주황색으로 설정
grid.col(0).set("style", "background", "orange");
// (0, 0)의 cell에 마지막으로 적용된 background 값은 "orange"이고, color는 "blue" 이다
var bgColor = grid.cell(0, 0).get("style", "background"); // orange
var color = grid.cell(0, 0).get("style", "color");        // blue

.alterStyle(styleName, styleValue)
이전에 .set()을 이용하여 설정해 놓은 값들 중 일부를 바꾸고 싶은 경우 .alterStyle()을 이용합니다. .set()과는 달리 이전에 적용된 상태를 유지하면서 특정 값만을 바꿀 수 있습니다.

// row의 배경색은 검정, 글자색은 파랑으로 설정
grid.row(0).set("style", {
  background: "black",
  color: "blue"
});
// column의 배경색을 주황색으로 설정
grid.col(0).set("style", "background", "orange");
// row의 background를 "green"으로 변경하지만 (0, 0)의 cell의 배경색은 "orange"로 유지됨
grid.row(0).alterStyle("background", "green");

Style 값 읽기

.get("style", [styleName])
.get() api를 이용하여 selector로 선택된 첫 번째 요소의 style 값을 구할 수 있습니다. 값을 구할 요소가 cell이라면 최종적으로 적용되는 style 값을 반환하며, 이외의 경우에는 사용자가 .set()이나 .alterStyle()을 이용하여 설정한 값을 그대로 반환합니다.
styleName이 주어졌다면 styleName에 해당하는 요소의 값을 반환하고, 그렇지 않다면 style 정보를 object로 반환합니다.

// row의 배경색은 검정 column의 배경색은 주황색으로 설정
grid.row(0).set("style", "background", "black");
grid.col(0).set("style", "background", "orange");
// (0, 0)의 cell에 마지막으로 적용된 background 값은 "orange"이다
var bgColor = grid.cell(0, 0).get("style", "background"); // orange
var obj = grid.cell(0, 0).get("style");
console.log(obj.background); // orange

Class 조작

Class 값 설정

.set("class", className)
.set() api를 이용하여 cell의 class 값을 설정할 수 있습니다. .set()을 이용하여 class 값을 설정하면 가장 마지막에 설정한 값이 적용됩니다.

grid.cell(0, 0).set("class", "A");   // 이 시점에서 cell의 class 값은 "A"이다
grid.cell(0, 0).set("class", "B C"); // 다시 .set()을 하면 cell에 class 값이 "B C"로 바뀐다

.addClass(className)
주어진 className을 기존 class의 값으로 추가합니다.

grid.cell(0, 0).set("class", "A"); // 이 시점에서 cell의 class 값은 "A"이다
grid.cell(0, 0).addClass("B");     // .addClass()를 하면 class 값이 "A B"로 바뀐다

.removeClass(className)
주어진 className을 요소의 class 속성에서 제거합니다.

grid.row(0).removeClass("classToRemove"); // 지정된 row에 포함된 모든 cell에서 "classToRemove"라는 class를 제거

.toggleClass(className, [switch])
주어진 className이 요소의 class 속성에 이미 적용되어 있다면 제거하고, 적용되어 있지 않다면 추가합니다. switch가 주어졌을 경우에는 이 값에 따라 추가나 제거합니다.

grid.col(0).toggleClass("toggle"); // column에 "toggle"이라는 class를 추가하거나 제거
gird.col(1).toggleClass("toggle", switch); // switch라는 변수의 값이 참이라면 class를 추가하고 그렇지 않다면 제거

Class 값 읽기

.get("class")
.get() api를 이용하여 selector로 선택된 첫 번째 요소의 class 값을 구할 수 있습니다. 값을 구할 요소가 cell이라면 최종적으로 적용되는 class 값을 반환하며, 이외의 경우에는 사용자가 .set()이나 .addClass()을 이용하여 설정한 값을 그대로 반환합니다.

grid.row(0).set("class", "A");
grid.col(0).set("class", "B");
var row = grid.row(0).get("class");   // "A"
var col = grid.col(0).get("class");   // "B"
var cell = grid.cell(0).get("class"); // "A B"

.hasClass(className)
특정 className이 selector로 선택된 첫 번째 요소에 적용되어 있는지 여부를 구할때 사용합니다.

var test = grid.hasClass("A"); // grid 전체 단위로 "A"라는 class가 적용되어 있는지 여부를 반환
