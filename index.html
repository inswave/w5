Lightweight, powerful and intuitive front-end library for easier and faster UI component development.
We first introduce one of our components, W5Grid.

Download this project as a .zip file Download this project as a tar.gz file

Features

Powerful

Powerful, what you need is here.
The W5Grid not only provides general grid functions but also infinite scroll, cross tab, grid in grid are available.

Highly adaptable

Highly adaptable.
Various UI components can be used as an inputType in GRID and data gets binding automatically.

Intuitive

Intuitive and easy to use.
Even with strong capabilities various functionalities, W5Grid is very intuitive and easy to use.

Customizable

Easy to customize.
The W5Grid has been developed with LESS, so that you can build your own Theme or even further develop your own style.

Display Variation

Support N-Screen.
Big data can be expressed in multi devices & multi platforms.
